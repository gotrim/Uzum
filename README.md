# Meu Repositório de Projetos em Vídeo

Este projeto foi criado para uso pessoal como um **reservatório de vídeos** de projetos que já fiz e que ainda estou desenvolvendo.  
Aqui você encontrará registros visuais do meu trabalho, permitindo acompanhar a evolução dos meus projetos e servir como portfólio pessoal.

## Funcionalidades

- Armazenamento organizado de vídeos de projetos.
- Registro histórico de projetos finalizados e em andamento.
- Referência pessoal para revisão e aprendizado contínuo.

## Tecnologias Utilizadas

- [MongoDB](https://www.mongodb.com/) (para armazenamento de dados, se aplicável)

## Observações

Este projeto é para uso pessoal e **não está aberto para uso público**, mas serve como registro e portfólio de aprendizado e evolução.
